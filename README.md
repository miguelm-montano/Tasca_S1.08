# Tasca_S1.08

Testing - Practice with PHPUnit

Use and practice with asserts, Data provider and TDD.

## Tools

- PHP
- PHPUnit
- Composer
- Visual Studio Code

## How to Install

Clone the repository:

```bash
   git clone https://github.com/miguelm-montano/Tasca_S1.08.git
```

On the terminal evaluate every Exercise with: phpunit folderOfTheExercise/tests (folderOfTheExercise: Exercise1, Exercise2, ExerciseN2_1, ExerciseN2_2, ExerciseN3)
